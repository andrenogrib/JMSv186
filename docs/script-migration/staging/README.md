# Staging Folder

This folder is reserved for future script imports after they are reviewed against the current BMS API.

Suggested future structure:

- `staging/npc/` for candidate NPC scripts
- `staging/quest/` for candidate quest scripts
- one script at a time, with a small markdown note explaining what was adapted

Reason for keeping this empty for now:

- the study shows several promising donor scripts
- but blindly copying many files at once would create avoidable regressions
- the current best path is curated import, not bulk replacement
